# LIFE NGO React + Tailwind

This project is migrated to React (Vite) and Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

## Pages

- `/` Home
- `/about` About Us
- `/what-we-do` What We Do
- `/contact` Contact

## Firebase

Firebase is configured through Vite environment variables in `.env` and used to load the hero image from Firestore path `Data/Images` field `Image_1`.
